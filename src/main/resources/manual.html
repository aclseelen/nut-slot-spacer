Metric or imperial? Click 'Inches' if your input values are in inches. The default is 'Millimeters'. All fields will be affected, except for the string diameters, that can only be entered in thousands of an inch.

Nut width: Enter the width of the nut (e.g. 1.6875)...

Edge spacing bass side: Enter the space between the outer string and the edge of the nut (e.g. .125)...

Edge spacing treble side: Do the same for the other side...

String diameters: Enter all string diameters in the right order (bass to treble), as a comma separated list. E.g.: 46, 36, 26, 17, 13, 10

Outer edge or center? Finally, choose whether the distance from the nut edges will be calculated to the outer string center or the outer edge. Default is `edge`.
